# Python File Manager

A simple command-line file management system built using Python.  
This project can create, read, update, rename, and delete files using terminal-based interaction.

## Features

- Create files
- Read file content
- Update existing files
- Rename files
- Append content to files
- Delete files
- Display files and folders

## Technologies Used

- Python
- pathlib module
- os module

## How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Python-File-Manager.git
```

2. Open the project folder

3. Run the Python file

```bash
python main.py
```
